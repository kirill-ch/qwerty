# Русская раскладка QWERTY для приложения слепой печати Type Fu

[Type Fu][1] - прекрасное полностью бесплатное приложение (и сайт) для обучения слепой печати.

Чтобы добавить русский язык вам потребуются 2 файла - русская раскладка и словарь слов.

Файл .tfl - русская раскладка QWERTY (ЙЦУКЕН).
Файл .tfd - словарь русских слов ["Новый частотный словарь русской лексики"][2].

В приложение Type Fu раскладка добавляется: Preferences -\> Keyboard -\> Import From File

Словарь добавляется: Preferences -\> Words -\> Import From File

Раскладку QWERTY я описал для Mac. 

Вы можете сами вносить изменения в текстовый файл раскладки, назначая клавиши клавиатуры нужным вам символам.
